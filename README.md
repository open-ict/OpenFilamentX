# OpenFilamentX

Open-source platform for decentralized filament production from recycled plastic.

## MVP Scope

- DIY filament extruder workflow
- Sensor ingestion for temperature, diameter, and speed
- FastAPI backend for telemetry and machine status
- Simple React dashboard scaffold
- Basic anomaly detection logic

## Stack

- ESP32 / Raspberry Pi
- FastAPI
- PostgreSQL-ready backend
- React + Vite
- Python AI utilities

## Repository Layout

- `hardware/` mechanical and electronics design placeholders
- `firmware/esp32/` example firmware for edge telemetry
- `backend/` API service
- `frontend/dashboard/` operator UI
- `ai/` anomaly detection utilities
- `docs/` build and architecture notes
