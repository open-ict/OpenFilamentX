# OpenFilamentX

Open-source platform for decentralized filament production from recycled plastic.

## Features

- DIY filament extruder
- IoT monitoring (temperature, diameter)
- Web dashboard
- AI-based optimization

## Stack

- ESP32 / Raspberry Pi
- FastAPI backend
- React frontend
- PostgreSQL
- PyTorch

## Goals

- Enable local manufacturing
- Reduce plastic waste
- Create open material datasets

## Status

MVP in development
